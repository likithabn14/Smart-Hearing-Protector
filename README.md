# 🛡️ Smart Hearing Protector

## Overview

Smart Hearing Protector is a Python Flask application that helps protect users from prolonged exposure to unsafe headphone volume levels. It continuously monitors the system audio level and automatically reduces the volume when the configured threshold is exceeded.

## Features

* 🎧 Real-time audio monitoring
* 🛡️ Automatic volume reduction
* 🎚️ Adjustable danger threshold
* 🔊 Configurable safe volume
* 🌐 Flask-based web interface
* ⚡ Live audio monitoring dashboard

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Pycaw
* COM API

## Note

The project uses Windows Audio APIs (`pycaw`). Audio monitoring behavior may vary depending on the Windows audio driver and Bluetooth device.
